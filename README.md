## The Vow - Build Instructions

## System Overview
The Vow is a Unity based game where the player explores the environment and collects items throughout the world.

## Repository Reference
The full project source code and deployment artifacts are available at:  
https://github.com/Ixchel-SB/Group-2---COMP-490-L

## Required Software
- Unity Hub installed
- Unity version: 6000.2.6f1
- Git (if cloning from the repository)

## Setup Instructions
1. Clone or download the project repository from GitHub.
2. Open Unity Hub.
3. Click Add Project.
4. Select the root folder of the downloaded repository.
5. Ensure the correct Unity version is selected.
6. Open the project.
7. In the Unity Editor:
- Navigate to Assets/Scenes/
- Open the main scene (MainMenu).
8. Press the Play button at the top of the Unity Editor to run the game.

## Validate the project:
1. Confirm the project opens in Unity without critical errors.
2. Confirm all required packages load successfully.
3. Open the main scene.
4. Press Play in the Unity Editor.
5. Confirm the game starts correctly.
6. Confirm the player can interact with the game environment.
7. Confirm there are no critical errors in the Unity Console.
