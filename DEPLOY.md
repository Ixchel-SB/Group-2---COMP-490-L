The Vow - Deployment Instructions

Repository Reference
The full project source code and deployment artifacts are available at:  
https://github.com/Ixchel-SB/Group-2---COMP-490-L

Deployment Overview
This Unity 3D game 'The Vow' can be deployed by creating a build from the Unity Editor.

Required Software
- Unity Hub installed
- Unity version: 6000.2.6f1
- Git (if cloning from the repository)

Build Instructions 
1. Clone or download the project repository from GitHub
2. Open Unity Hub
3. Click Add Project
4. Select the root folder of the downloaded repository
5. Ensure the correct Unity version is selected
6. In Unity, go to: File > Build Settings
7. Select the target platform
8. Make sure the main game scene is included in Scenes In Build
9. Click Build
10. Choose a build output folder outside the Unity project folder

Deployment Instructions
1. Locate the generated build folder
2. Keep the executable and supporting files together
3. Share or upload the full build folder
4. Run the executable to launch the game

Deployment validatation:
- The game launches successfully
- The main scene loads correctly
- Player controls work as expected
- Core gameplay features function correctly
- No critical runtime errors occur
