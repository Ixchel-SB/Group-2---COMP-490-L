## The Vow - Validation

# Repository Reference
The full project source code and deployment artifacts are available at:  
https://github.com/Ixchel-SB/Group-2---COMP-490-L

# Validate checklist:
1. Confirm the project opens in Unity without critical errors.
2. Confirm all required packages load successfully.
3. Open the main scene.
4. Press Play in the Unity Editor.
5. Confirm the game starts correctly.
6. Confirm the player can interact with the game environment.
7. Confirm there are no critical errors in the Unity Console.

# In-Game checklist:
1. Player can move and explore world
2. Player can interact with items and NPC's
3. Notes/clues can be read from inventory
4. Audio plays correctly
5. Scenes load correctly
6. Game pauses with 'P' key
7. Pause menu opens with 'P' key
